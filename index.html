Senior Thesis · Phase 1

Luna Yoga

A personalized yoga companion app built from scratch as a senior thesis project at UC Santa Cruz. Phase 1 covers the full frontend architecture and a Node.js backend with user authentication, session tracking, and a pose recommendation engine. Designed with accessibility and flow in mind — Luna meets you wherever you are in your practice.

React Native · Node.js · Supabase 2025 Solo Project
01 / 04

Login Screen

A calm, minimal entry point where users sign in to their personalized yoga journey. Supports email/password auth with secure token handling on the backend. First impressions matter — the login sets Luna's meditative tone from the start.

02 / 04

Collective

The heart of the app — a personal dashboard showing weekly streaks, completed sessions, and AI-curated recommendations based on your history. Designed to motivate without overwhelming, using soft data visualizations and gentle nudges.

03 / 04

Settings

Full user control over experience preferences — notification timing, difficulty level, preferred yoga styles, and accessibility options. Clean toggle-based UI that respects user autonomy and keeps customization friction-free.

04 / 04

Yoga Types

An explorable library of yoga disciplines — from restorative Yin to dynamic Vinyasa — each with descriptions, difficulty ratings, and curated session paths. Users can favorite styles to shape their recommendation feed.
